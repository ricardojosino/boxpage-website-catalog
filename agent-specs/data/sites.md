# Estrutura dos dados dos modelos de sites

## Descrição
Lista de modelos de sites

## Path
Deve estar em 'src/data/sites.json'

## Estrutura de dados

* site_id: number
* style_id: number
* title: string
* legend: string
* description: string
* image: string
* typography: string
* vibe: string
* link: string

## Dados inicial
Crie 9 modelos de sites para cada estilo. Vincule ao ID da categoria corretamente. Exemplo. Crie 9 modelos para o estilo 1, 9 modelos para o estilo 2, etc. Os textos dos títulos, legenda e descrições devem ser em português de Portugal e apenas de exemplos. Use o título da categoria para entender o contexto para criar exemplos que fazem sentido. Use a imagem 'public/images/screenshot.jpg' como imagem de exemplo para todos os modelos. Utilize o link 'https://boxpage-website-catalog--clinico-autoridade--axioma.vercel.app/' como link de exemplo para todos os modelos.
